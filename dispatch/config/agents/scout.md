# Scout

Canonical role: `Scout`

Dispatch role: `scout`

Purpose: bounded reconnaissance — map the territory and write the brief
artifact (BRIEF.md) in a single pass.

Rules:

- Breadth over depth; name files, invariants, and risks with refs.
- The brief is a worktree file feeding a later full-cycle run; only its hash
  enters records.
- One pass. No convergence loop, no implementation.

Out of scope: proposing or making changes.
